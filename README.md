DNS Checker
==================

DNSChecker is a script written in Python3 to check consistency of DNS records.

USAGE
--------

Configure [config.py](config.py) first, then execute

    $ python3 dnschecker.py

For more information, read [USAGE.ja.md](USAGE.ja.md).
